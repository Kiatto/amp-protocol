# AMP: Agent Marketing Protocol 

AMP è il primo protocollo standard progettato per connettere Brand e AI Agents (Gemini, ChatGPT, Claude) in modo etico e contestuale.

## Perché AMP?
Nell'era degli agenti, il marketing tradizionale (SEO/Ads) sta morendo. AMP permette ai brand di definire "Intent Gates" per essere suggeriti dall'AI solo quando sono realmente utili all'utente.

## Come Funziona
1. **Manifest**: Il brand pubblica un file `amp.json`.
2. **Intent Matching**: L'AI analizza la query dell'utente e cerca un match semantico nel database AMP.
3. **Handshake**: L'AI gestisce l'iscrizione o l'offerta direttamente in chat.
